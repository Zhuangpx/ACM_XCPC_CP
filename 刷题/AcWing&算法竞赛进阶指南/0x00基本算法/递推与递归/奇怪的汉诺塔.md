# 奇怪的汉诺塔

一个n盘四塔问题。

先讨论三塔问题。运用递推的思想，需要移动第n个先要移动前n-1个，具体为：

(1,n-1)--->B     [n]---->C        (1,n-1)----->C

因此有：

```c++
d[n]=d[n-1]*2+1
```

递推一下得出答案。

那么四塔类似有：

四塔：(1,i)---->B

三塔：(i+1,n)---->C

四塔：(1,i)------>D

最终实现n----->D

其中i-----[1,n]，即i枚举1到n更新最小值。

最后有：

```c++
f[i]=min(f[i],f[j]*2+d[n-j]);//i表示当前一共有几个塔，也就是上文所说的n。
```

类似地，可以推广到m塔问题。

```c++
#include <bits/stdc++.h>
using namespace std;
int d[21],f[21],i,j;
int main()
{
    for (i=1;i<=12;i++)
        d[i]=2*d[i-1]+1;
    memset(f,0x3f,sizeof(f));
    f[0]=0;
    for (i=1;i<=12;i++)
        for (j=0;j<i;j++)
            f[i]=min(f[i],f[j]+f[j]+d[i-j]);
    for (i=1;i<=12;i++)
        cout<<f[i]<<endl;
}
```
